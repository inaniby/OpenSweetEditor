# Demo

桌面端 Avalonia 示例工程。

## 功能

- 大文档加载与切换
- Decorations：syntax / inlay / diagnostic / fold / gutter / phantom / guides
- Completion popup
- Inline suggestion
- Snippet / linked editing
- New line action provider
- Selection menu custom actions
- Theme / wrap / perf overlay / keymap 切换

## 构建

```bash
cd platform/Avalonia/Demo
# 需要 .NET 8 SDK 与 Avalonia 11 依赖
```
